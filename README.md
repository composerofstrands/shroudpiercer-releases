# Shroudpiercer
**Shroudpiercer** is a Stellaris companion desktop app that reveals the real tech deck.
Stellaris uses a weighted draw system for technologies.
Shroudpiercer calculates the full eligible pool and actual draw probabilities
based on your empire setup and research state.

## Downloads
**Download the latest alpha build https://github.com/composerofstrands/shroudpiercer-releases/releases/latest**

## Compatibility
- Stellaris: **4.2.4**
- Shroudpiercer: **v0.2.0-alpha**

## Features (Alpha)
- Configure empire setup (ethics, civics, origin)
- View all currently eligible technologies
- See effective draw weights and probabilities
- Mark technologies as researched to update the pool in real time

## Alpha Status
This is an early alpha release.
Expect bugs, incomplete features, and frequent updates.
Save files and UI may change between versions.
